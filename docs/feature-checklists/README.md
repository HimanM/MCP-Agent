# Feature Checklists

This folder tracks the staged feature work for the Kapruka challenge app.

## Planned iterations

- `01-gift-advisor.md` - guided gift selection flow
- `02-order-tracking.md` - order tracking tool, prompt flow, and tracking card UI
- `03-pay-link-card.md` - post-checkout order summary and payment-link card
- `04-voice-mode.md` - browser-native voice input and spoken assistant replies
- `05-quick-actions-and-welcome.md` - richer quick actions, onboarding, and category-first prompts
- `06-multilingual-ui.md` - localized UI copy and language-aware helper flows
- `07-reliability-and-debug-surface.md` - provider fallback, MCP debug route, and light request hardening

## Working rule

Each file is a per-feature checklist. We update the relevant file as a feature moves from planned to shipped.
